# TUPT Environmental Data Scripts

Welcome to my repository dedicated to atmospheric and environmental data analysis. As an incoming BS Environmental Science student at the Technological University of the Philippines Taguig (TUPT), this space serves as my playground for learning geospatial analysis, automated script development, and climate reporting systems.

---

## Projects Overview

###  1. Air Quality Index (AQI) Evaluator (`aqi_checker.py`)
A Python script that evaluates local ambient air conditions in Taguig, Metro Manila, to check if parameters fall within safe regulatory thresholds.
* **Core Logic**: Evaluates specific AQI numbers against public health safety parameters.
* **Future Extension**: Integrate real-time API fetches from OpenWeather or localized sensor nodes.

###  2. Precipitation Maxima Calculator
A lightweight statistical script designed to process localized precipitation arrays, locating maximum extreme rainfall values for environmental field planning.
